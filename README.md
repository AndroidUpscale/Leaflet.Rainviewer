# Leaflet.Rainviewer

E.g.

````js
    // Change default options
    L.control.rainviwer({ 
        position: 'bottomleft',
        nextButtonText: '>',
        playStopButtonText: 'Play/Stop',
        prevButtonText: '<',
        positionSliderLabelText: "Godzina:",
        opacitySliderLabelText: "Przezroczystość:",
        animationInterval: 500,
        opacity: 0.5
    }).addTo(map);
````
